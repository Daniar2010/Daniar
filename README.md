# Random Quote Generator

**Автор:** Данияр Маннапов   
**Дата:** Апрель 2026

## Описание

Простое приложение с графическим интерфейсом на Python (tkinter) для генерации случайных цитат. Поддерживает:

- Случайный выбор цитаты из предопределённого списка или добавленных пользователем.
- Сохранение истории сгенерированных цитат в JSON.
- Фильтрацию истории по автору и теме.
- Добавление новых цитат с проверкой на пустые поля.
- Загрузку и сохранение истории между сеансами.

## Требования

- Python 3.6+
- Стандартные библиотеки: tkinter, json, random, os, datetime
